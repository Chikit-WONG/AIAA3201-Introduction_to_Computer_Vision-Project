Abstract

This report studies automatic video object removal under the AIAA3201 project setting. We build three progressively stronger pipelines: a hand-crafted baseline that combines YOLOv8-Seg, sparse optical-flow motion filtering, temporal background propagation, and OpenCV inpainting; an AI-driven reproduction using SAM 2 or VGGT4D masks with ProPainter; and an exploration stage using SAM 3 masks with ProPainter, DiffuEraser, and ROSE inpainting backends. We evaluate mask quality on all 90 DAVIS validation sequences using Jaccard mean (JM) and Jaccard recall (JR), and evaluate video restoration on paired Wild Video clips using PSNR and SSIM. SAM 2 is a strong Part 2 baseline (JM 0.9167, JR 0.9846), while the SAM 3 multi-object setup improves DAVIS mask quality to JM 0.9423 and JR 0.9952. The best Wild Video PSNR is achieved by SAM 3 + DiffuEraser, while SAM 3 + ROSE-SE yields the best SSIM.

Pipeline Overview

All three parts share the same high-level structure: extract frames → estimate or prompt object masks → optionally refine masks → inpaint the masked regions → evaluate outputs. They differ mainly in the mask source and inpainting backend.

Fig. 1  —  Unified pipeline roadmap across the three project parts. The same frame, mask, inpainting, and evaluation interfaces make the classical, reproduced, and exploratory pipelines directly comparable.
PART 1

Hand-Crafted Baseline

  • YOLOv8-Seg instance segmentation
  • Lucas-Kanade sparse optical flow filtering
  • Morphological mask dilation
  • Farneback flow-aligned temporal background propagation
  • Telea / Navier-Stokes cv2.inpaint fallback
PART 2

AI-Driven Reproduction

  • Pipeline A: VGGT4D masks + ProPainter
  • Pipeline B: SAM 2 (first-frame prompt) + ProPainter
  • Evaluated on all 90 DAVIS validation sequences
  • SAM 2 JM 0.8383 on mandatory sequences
PART 3

Optimization & Extension

  • SAM 3 multi-object tracker
  • Side-effect mask: shadows & reflections
  • Backends: ProPainter, DiffuEraser, ROSE
  • Best JM 0.9423 / JR 0.9952 on DAVIS-90

Qualitative Results

Each triptych shows a sample frame: input with person (left) → predicted mask (center) → inpainted output (right).

Part 1 (Hand-Crafted) — bmx-trees sequence. Temporal + flow-alignment achieves masked PSNR 16.84 dB, up from 14.00 without alignment.
Part 2 (SAM 2 + ProPainter) — strong mask quality (JM 0.84) yields clean ProPainter inpainting on the tennis sequence.
Part 3 (SAM 3 + generative backends) — SAM 3 multi-object initialization improves tracking stability; ROSE and DiffuEraser provide diffusion-based restoration.
Failure case: rhino sequence — fast motion and complex background texture cause tracking drift and inpainting artifacts.

Related Work

Key external models and datasets used in this project:

  • YOLOv8-Seg — Ultralytics, real-time instance segmentation
  • SAM 2 — Ravi et al., ICCV 2024 — promptable video segmentation
  • SAM 3 — Meta AI, 2025 — concept-level video segmentation
  • VGGT4D — Hu et al., arXiv 2024 — 4D scene reconstruction with motion cues
  • ProPainter — Zhou et al., ICCV 2023 — propagation-based video inpainting
  • DiffuEraser — Li et al., 2024 — diffusion-based video inpainting
  • ROSE — Kunbyte AI, 2024 — transformer video inpainting
  • DAVIS 2017 — Pont-Tuset et al. — video object segmentation benchmark
